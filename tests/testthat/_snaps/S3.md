# S3_class() checks its inputs

    Code
      S3_class(1)
    Error <simpleError>
      `class` must be a character vector

    Code

      First argument to `constructor` must be .data
    Code
      S3_class("foo", function(.data, ...) { })
    Error <simpleError>
      `constructor` can not use `...`
